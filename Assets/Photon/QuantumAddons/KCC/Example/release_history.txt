# Photon Quantum KCC Addon Sample

## 3.0.5

- Added Moving Platform processor.
- Improved Teleport processor - now teleports to a specific destination.
- Improved Jump Pad processor - now applying dynamic velocity to reach target point.
- New Showcase scene with modular environment and processor examples - gravity zone, attractor, portals, moving platforms.
- Added Quantum Demo Input package. The Quantum input struct (Quantum.Input) is now defined by QuantumDemoInputBase.qtn from this package.
- The KCC sample scripts now work with BasePlayerInput - a custom struct which is converted from/to Quantum.Input struct.
- Ground is now made of smaller colliders - this fixes FP precision issues when making sphere casts against ground.
- Added penetration visualizer for debugging purposes.
- Updated KCC addon to 3.0.5.
- Updated Quantum SDK to 3.0.2 Stable 1660.
- Updated Quantum DemoInput package to 3.0.2 Stable 1660.
